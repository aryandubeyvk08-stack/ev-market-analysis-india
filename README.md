# 📊 EV Market Analysis & Adoption Trends in India

## 🚀 Overview

This project analyzes Electric Vehicle (EV) adoption patterns across India using real-world data to extract business insights.

---

## 📈 Key Insights

* Three-wheelers dominate EV adoption (~67%), driven by commercial usage
* Two-wheelers (~29%) show affordability-driven growth
* EV adoption strongly correlates with charging infrastructure availability
* Significant opportunity exists in expanding EV infrastructure

---

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib

---

## 📊 Analysis Performed

* State-wise EV adoption trends
* Vehicle category segmentation
* Charging infrastructure distribution
